# Folder for generated Images
Please put generated Images in here.
Create one subfolder per image generator
```
generated_images
└── GeneratorA
    ├── README.md
    └── images
        ├── 0001.png
        └── ...
└── GeneratorB
    ├── README.md
    └── images
        ├── 0001.png
        └── ...
...
```
